# sisyffes-tinspire-programs
A Repository containing python scripts I wrote on my TI-Nspire CX II-T calculator (in french because I am)

## Specifications
- These programs are designed to be ran on **python 3.4.0**. You may change thing if you intend to run this on micropython.
- These programs are built to run on a c**onsole 11 lines tall and aprox 35 dashes long** (it depends on the character, the font on the calculator is not monospace).
- The programs are built with the **keyboard layout of the calculator** in mind.
- All the user interfaces of the programs are **written in French**, consider using a tool like [deepl.com](deepl.com) to translate.

## Scripts included
in `programs/` :

- `calcul_molaire.py` : equilibrate chemical reaction, get the molar mass of chemicals...
- `decomposition.py` : decompose any natural integer to a product of prime factors.
- `statistiques.py` : work with a simple dataset and calculate averages, median, standart deviations...
- `suites.py` _not done_ : work with reccurent or explicit mathematical suites.
- `vecteurs.py` : add, subtract, dot product vectors.
